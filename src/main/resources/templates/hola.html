El fondo se configura desde application.properties con la clave app.bg.

Cómo funciona `app.bg`

Es el valor CSS que se aplicará a background del body. Puede ser cualquier valor válido de CSS, por ejemplo gradientes:

  • linear-gradient(135deg,#eef2ff 0%,#fef3c7 100%) — degradado lineal.
  • radial-gradient(circle at 30% 30%, #c1ffd7 0%, #f0fff4 60%) — degradado radial.
  • #ffffff — color sólido también válido.

Ejemplo en application.properties: 

app.bg=linear-gradient(135deg,#f0f9ff 0%, #fff1f2 100%)

Después de cambiarlo, reinicia la aplicación para que se aplique el nuevo fondo.
